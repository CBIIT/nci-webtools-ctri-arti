PARTICIPATING SITE (pSITE) PROTOCOL ADDENDUM TEMPLATE FOR USE WHEN NIH IS THE REVIEWING IRB FOR AN EXTERNAL INSTITUTION/SITE

This preface contains clarification about the purpose of this template and should be removed before it is finalized and distributed.

The pSITE Protocol Addendum is required as a supplement to the pSITE Application in the PROTECT eIRB system and provides the NIH IRB with important information about how the protocol will be implemented at the pSITE.

The goal of this template is to assist the pSITE Principal Investigator in communicating pSITE local context information to the NIH IRB and to allow for a more detailed description of how the research will be implemented at the pSITE, particularly when it will differ to how the research is described in the main protocol.

This document needs to be submitted with the initial pSITE submission in PROTECT as a standalone document.

Style Guidelines:

12-point font (Times, Times New Roman or Arial) and single line spacing for the protocol addendum document. Section numbers shown refer to sections in the main study protocol. Margins should be 1 inch on all sides.  This document is already formatted using headings in the styles.

Instruction/explanatory text is indicated by italics and should be deleted prior to finalizing the addendum. This text provides information on the content that should be included in the addendum.

Version control is important to track protocol development, revisions, and amendments. With each revision, the version date located in the header of each page needs updating.

If a section is not applicable to the pSITE, do not delete it; rather indicate that the section is not applicable. Duplication of information in the protocol addendum that is already in the main protocol should be avoided.
